# Web Recon Tool

A Python-based reconnaissance tool that automates the **information gathering phase** of penetration testing and provides **actionable insights**, not just raw data.

---

## 🚀 Key Features

- Subdomain Enumeration (Passive + Brute-force)
- Live Host Detection (HTTP probing)
- Port Scanning (Async + Banner Grabbing)
- Technology Fingerprinting (Headers + Patterns)
- Endpoint Discovery (Crawler + Wayback)
- JavaScript Analysis (API endpoint extraction)
- DNS Records & Email Security Checks
- Security Headers Analysis
- CVE Mapping (Filtered & Relevant)
- Risk Scoring Engine (Prioritized Output)
- Professional HTML Report Generation

---

## 🧠 Why this tool?

Most recon tools dump data.  
This tool focuses on:

- 🔍 **Prioritization** (HIGH / MED / LOW endpoints)
- 🎯 **Actionable findings**
- ⚠️ **Risk-based decision making**
